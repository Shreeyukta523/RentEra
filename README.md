# 🏠 RentEra – AI-Powered Secure Rental Management Platform

Rentera is an AI-powered, secure rental management platform that connects **Property Owners (Proprietors)** and **Renters** through intelligent automation, verified identities, and seamless digital payments.

The platform simplifies the rental process by combining **Artificial Intelligence, Natural Language Processing (NLP), secure authentication, property verification, and digital payment integration** into one trusted ecosystem.

---

# 🚀 Features

## 🤖 AI Chatbot & Intelligent Assistance

Rentera places Artificial Intelligence at the center of the user experience to make property searching and rental management faster and smarter.

### 🧠 NLP-Powered AI Chatbot

The chatbot understands natural language and helps users find properties effortlessly.

### Key Features

* Natural language property search
* Smart query understanding
* Context-aware conversations
* Instant responses
* Property inquiry assistance
* Payment guidance
* Listing navigation

### Example

**User**

> Show me a 1BHK under Rs. 12,000 near the city center.

**Rentera AI**

> Here are 4 verified properties matching your criteria.

# 🔐 Security & Verification

Rentera prioritizes security and trust by implementing multiple verification layers.

## User Authentication

* Email verification
* Secure login
* OTP-based password recovery
* Secure session management
* Encrypted user data

---

## 🏡 Proprietor Verification

Before listing any property, proprietors must complete identity verification.

Verification includes:

* Citizenship verification
* Land ownership verification (Lalpurja / Land Number)
* Email verification
* OTP confirmation

This helps eliminate fake property listings and fraudulent users.

---

## 🧑‍💼 Renter Verification

Renters are also verified to ensure safe communication and transactions.

Verification includes:

* Email verification
* OTP authentication
* Identity validation

Only verified users can rent or communicate through the platform.
